# Анализ конфиденциальных данных компании "Медикаменте"

## Выявленные конфиденциальные данные

### 1. Персональные данные пациентов (PII - Personally Identifiable Information)

#### Базовые персональные данные:
- ФИО (фамилия, имя, отчество)
- Дата рождения
- Телефон
- Электронная почта

#### Расширенные персональные данные:
- Адрес прописки
- Место работы или учёбы
- Хронические заболевания
- Медицинская история

### 2. Медицинские данные (PHI - Protected Health Information)

#### Медицинские карты:
- Диагнозы
- Результаты анализов
- Заключения врачей
- Назначенные препараты
- История лечения

#### Результаты анализов:
- Лабораторные исследования
- Инструментальные исследования
- Биометрические данные

### 3. Финансовые данные

#### Платежная информация:
- Данные о платежах
- Счета и квитанции
- Договоры на оказание услуг
- Финансовая история пациента

### 4. Административные данные

#### Учётные записи:
- Данные о записях к специалистам
- Журналы приёма пациентов
- Расписание работы специалистов

### 5. Данные сотрудников

#### Кадровые данные:
- Персональные данные сотрудников
- Данные о зарплате
- Налоговая информация

## Текущее состояние хранения данных

### Проблемные зоны:

1. **Excel файлы на общем диске**
   - Журналы приёма пациентов
   - Учёт пациентов
   - Учёт платежей
   - Медицинские карты (частично)

2. **Сканы документов**
   - Медицинские карты
   - Договоры
   - Результаты анализов

3. **1С:Бухгалтерия предприятия**
   - Учёт денежных средств
   - Интеграция с ККМ
   - Финансовые данные

4. **1С:Торговля и склад**
   - Учёт ТМЦ
   - Закупки оборудования

5. **Физический сервер**
   - Active Directory
   - Файловый сервер
   - Exchange Mail Server

## Классификация данных по уровню конфиденциальности

### Критический уровень (требует максимальной защиты):
- Медицинские диагнозы
- Результаты анализов
- Хронические заболевания
- Финансовые данные пациентов

### Высокий уровень:
- ФИО, дата рождения
- Адрес прописки
- Контактные данные
- История лечения

### Средний уровень:
- Данные о записях
- Расписание специалистов
- Общая информация о пациентах

### Низкий уровень:
- Агрегированная статистика
- Обезличенные данные для аналитики
